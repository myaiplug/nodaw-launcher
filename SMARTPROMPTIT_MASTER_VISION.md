## 🧠 THE TEN AI MASTERY PROFILES

Each AI has unique strengths, quirks, and optimal prompting patterns. SmartPromptIt contains **deep behavioral models** for each:

### 1. **ChatGPT (GPT-4/4o/4.5)**
**Personality:** Helpful, verbose, occasionally sycophantic
**Optimal Strategy:** 
- Use role-playing ("Act as a...")
- Request step-by-step reasoning
- Ask for multiple perspectives
- Specify output format explicitly
- Use System-like instructions even in user prompts

**Prompt Template Architecture:**
```
[ROLE DEFINITION]
[CONTEXT ESTABLISHMENT]
[SPECIFIC TASK WITH CONSTRAINTS]
[OUTPUT FORMAT SPECIFICATION]
[QUALITY GATES / SELF-CHECK INSTRUCTIONS]
```

**Example Enhancement:**
```
Before: "Write a poem about rain"
After:  "You are a poet laureate with mastery of both classical and modern forms. 
        Write a poem about rain that: 1) Uses imagery appealing to at least 3 
        senses, 2) Contains one unexpected metaphor, 3) Has a subtle emotional 
        arc from melancholy to hope, 4) Is exactly 14 lines (Shakespearean sonnet 
        structure but with modern language). After writing, explain your artistic 
        choices for each stanza."
```

## 💎 ENHANCEMENT TECHNIQUES LIBRARY

### 1. Role Priming
Transform generic requests by establishing expert personas:
```
WEAK:   "Fix this code"
ELITE:  "Act as a senior software architect specializing in performance 
        optimization. Review this code for: 1) Algorithmic efficiency, 
        2) Memory management, 3) Code readability, 4) Potential edge cases. 
        Provide fixes with explanations for each improvement."
```

### 2. Constraint Injection
Add boundaries that improve output quality:
```
WEAK:   "Write a story"
ELITE:  "Write a short story (800-1000 words) with the following constraints: 
        1) Exactly 3 characters, 2) Single location, 3) Takes place in real-time, 
        4) Must include one plot twist in the final paragraph, 5) Written in 
        present tense, 6) No dialogue tags other than 'said'."
```

### 3. Output Format Specification
Demand structured responses:
```
WEAK:   "Compare React and Vue"
ELITE:  "Compare React and Vue frameworks. Structure your response as:
        
        ## Quick Verdict (2 sentences)
        ## Comparison Table
        | Aspect | React | Vue | Winner |
        ## Deep Analysis (500 words)
        ## When to Choose Each
        ## Code Example: Same Component in Both"
```

### 4. Chain-of-Thought Triggering
Force reasoning transparency:
```
WEAK:   "Solve this math problem"
ELITE:  "Solve this math problem. Walk through your reasoning step-by-step, 
        showing all intermediate calculations. After solving, verify your 
        answer using an alternative method. If you notice any errors during 
        verification, correct them and explain what went wrong initially."
```

### 5. Perspective Multiplication
Request diverse viewpoints:
```
WEAK:   "Should I start this business?"
ELITE:  "Evaluate this business idea from 5 distinct perspectives:
        1) Optimistic investor looking for growth potential
        2) Skeptical veteran entrepreneur identifying risks
        3) Target customer evaluating value proposition
        4) Competitor strategist assessing market threats
        5) Your own balanced analytical view
        Conclude with a synthesis of the most important factors."
```

### 6. Quality Gate Insertion
Add self-verification instructions:
```
WEAK:   "Write unit tests"
ELITE:  "Write comprehensive unit tests for this function. Requirements:
        1) Cover all code paths (aim for 100% branch coverage)
        2) Include edge cases (null, empty, boundary values)
        3) Test error handling explicitly
        4) Use descriptive test names that explain the scenario
        5) After writing, review your tests and identify any missing scenarios
        6) Estimate the confidence level of your test coverage (1-10)"
```
