# Fonts Directory

Place the following font files here:

- `Inter-Regular.ttf` - Main UI font
- `Inter-Bold.ttf` - Headings and labels
- `JetBrainsMono-Regular.ttf` - Numeric values and code

Download links:
- Inter: https://fonts.google.com/specimen/Inter
- JetBrains Mono: https://www.jetbrains.com/lp/mono/
